# bosonicplus
Extension of the bosonic backend from strawberryfields that works. 

## Requirements
- Python 3.11.3 (`scipy==1.9.3` for `strawberryfields`)
## Installation
`pip install -e .` from root.

## Tutorial
See notebooks in demos
